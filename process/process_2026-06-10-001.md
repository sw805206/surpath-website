# process.md

## Purpose
Project-specific process rules for surpath-website. Always read process/claude.md first, then this file. If they conflict, stop and ask.

## Project-Specific Rules

### Deployment
Cloudflare Pages watches main. Merge auto-triggers deployment. No separate publish step.

### Style Consistency
On a regular basis, user manually triggers a style-consistency review as best practice, especially after significant UI changes.

## Shared Stylesheet

All pages must link to `style/style.css` as the single shared stylesheet. Never embed styles in individual HTML files. When adding new style patterns, add them to style/style.css and document the values in style/style.md before building the page.
